# Contributing

OpenPhone contribution terms and review expectations live in repository
metadata so pull requests, issue templates, and the docs site point to the same
source of truth.

- [Contribution terms](https://github.com/secondly-com/OpenPhone/blob/main/.github/CONTRIBUTING.md)
- [Security policy](https://github.com/secondly-com/OpenPhone/blob/main/.github/SECURITY.md)

For local work, run the repository checks before opening a pull request:

```bash
./scripts/check.sh
git diff --check
```
